## 🔍 なぜ移行したか(2026-05-12)

PR Agent が本体から独立リポジトリ化された(`shiwake-ai-PR-Agent`)ことに伴い、セッションログも PR Agent 側で管理することにした。

理由:
- 直近のセッションは PR Agent 関連の作業が中心
- 本体側のファイル混在を解消する目的と一致
- PR Agent 側で独立した歴史を作る

ただし、本体側でも作業した内容(検査レポート対応・本体機能追加など)は引き続き本ポインタ経由で参照可能。
