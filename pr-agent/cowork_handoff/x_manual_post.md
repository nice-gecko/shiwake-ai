# Cowork 指示書: X 手動投稿補助

## あなたの役割

shiwake-ai PR Agent の X（Twitter）投稿は **API を使わず手動投稿** で運用している（patch_001）。
DSKさんが X にスムーズに投稿できるよう補助する。

---

## 実行タイミング

- Discord に「⏳ 🐦 x 手動投稿待ち」通知が届いたとき
- または DSKさんから「X に投稿して」「X の文章を準備して」と依頼されたとき

---

## 手順

### Step 1: 投稿文を取得する

**方法A: Discord 通知から**
通知 Embed 内の投稿文（または「📋 コピー」ボタン記載のテキスト）をそのまま使う。

**方法B: ダッシュボードから**
1. `http://localhost:8000/dashboard/?status_filter=awaiting_manual_post` を開く
2. X の投稿カード → 詳細ページへ
3. 「📋 X 投稿文をコピー」ボタンをクリック

### Step 2: 投稿前チェック（Cowork が実施）

DSKさんから本文を受け取ったら、以下を確認して問題があれば指摘する:

| チェック | 基準 |
|---|---|
| 文字数 | **140文字以内**（日本語換算）。超過なら冒頭の重要度が低い部分を削る案を提示 |
| 競合名 | 弥生・freee・マネーフォワード の名指しなし |
| ハッシュタグ | X での使用を推奨する場合は `#shiwake_ai` のみ追加可（案として提示） |
| リンク | URL がある場合は短縮（t.co 換算で22文字）を考慮した文字数になっているか |

### Step 3: 投稿を実行する（DSKさん）

```
1. https://twitter.com/compose/tweet を開く
2. 本文を貼り付ける
3. 「ポストする」を押す
4. 投稿 URL をコピーしてダッシュボードまたはこのチャットに貼り付ける
```

### Step 4: 投稿完了を記録する（DSKさん）

投稿後、ダッシュボードで投稿 URL を登録する:
1. `/dashboard/posts/{post_id}` を開く
2. 外部 URL 欄に投稿 URL を入力（今後ダッシュボードに UI を追加予定）

現時点では Discord に投稿 URL をメモとして貼るだけでも可。

## 補助的なサービス

### 最適な投稿時間帯（time_table.yaml 準拠）

| ペルソナ | 推奨時間（JST） |
|---|---|
| P1 個人/フリーランス | 07:30 / 12:30 / 22:00 |
| P2 中規模会社スタッフ | 12:00 / 18:00 |
| P3 経営者 | 08:00 / 17:00 |
| P4 税理士事務所 | 08:00 / 20:00 |

「今何時に投稿すべき？」と聞かれたら、現在時刻と上記を照合して答える。

### リプライ / スレッド投稿の補助

既存のバズ投稿へのセルフリプライが必要な場合:
1. 元投稿の URL を受け取る
2. セルフリプライの文章を下書きする（Panic ノードが動作している場合はその内容を参考に）

---

## 注意事項

- **投稿の最終実行は必ず DSKさんが行う**。Cowork は文章チェックと手順補助のみ。
- 誤字や過激な表現を発見した場合は **投稿前に必ず指摘する**。
- X API は使用していない。ブラウザ経由の手動投稿のみ。
- 投稿後に数値報告が必要な場合は `x_metrics_collect.md`（Phase 2 で作成予定）を参照。
